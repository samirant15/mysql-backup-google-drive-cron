# MySQL backup to Google Drive Cron Job

Simple cron job that creates a backup from a MySQL DB and uploads it to Google Drive

## Getting Started

1. 📦 Install dependencies
```
npm install 
```
or
```
yarn install 
```

2. ⚙️ Configure .env file

3. 🚀 Start cron job
```
yarn build 
```
then
```
yarn start 
```
